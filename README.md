#Scanning Barcodes with iOS 7
============================+

## Overview

With the release of iOS 7, Apple included the ability to detect barcodes through its AV Foundation framework. The supported types are:

* UPC-A
* UPC-E
* Code 39
* Code 39 mod 43
* Code 93
* Code 128
* EAN-8
* EAN-13
* Aztec
* PDF417
* QR
